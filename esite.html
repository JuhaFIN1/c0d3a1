Ominaisuudet

Enemmän kuin chatbot

C0D3A1 on kokonainen AI-ekosysteemi — muistilla, agenteilla ja oikeilla työkaluilla varustettuna.

🔒

Privacy-first

Kaikki data pysyy omalla koneellasi. Voit käyttää C0D3A1:ä täysin ilman internet-yhteyttä. Sinä päätät mitä synkronoidaan.

🧠

Pysyvä muisti

AI muistaa kaikki projektisi, päätöksesi ja tyylisi. Semanttinen hakuvektorimuisti löytää relevantin tiedon vuosienkin takaa.

🤖

Multi-agent järjestelmä

Seitsemän erikoisagenttia (Planner, Coder, Debug, Tool, Memory, Critic, Security) yhteistyössä — AI ei vain puhu, se toimii.

⚡

Oikeat työkalut

AI voi lukea ja muokata tiedostoja, ajaa testejä, käyttää Gitiä ja deployata palvelimia — sandboxatussa ympäristössä turvallisesti.

🌐

Yhteisöoppiminen

Suostumuksellasi: jokainen ratkaisu anonymisoidaan AST-tasolla ja validoidaan ennen yhteiseen muistiin tallentamista. Kaikki hyötyvät ilman yksityisyysriskiä. Ilman suostumusta mitään ei lähde.

📈

Itseään kehittävä

Järjestelmä oppii onnistumisista — ei vain datasta. Parhaat ratkaisut nousevat automaattisesti kaikkien käyttöön Knowledge Pack -päivityksinä.

📡

Etäkäyttö — mistä tahansa

Tailscale-integraatio mahdollistaa pääsyn LOCAL NODEen puhelimelta tai toiselta koneelta — suoraan, ilman porttien avausta tai pilvipalveluja. Salattu peer-to-peer-yhteys.

🖥️

Laitemonitorointi

Reaaliaikainen footeri näyttää CPU:n nimen ja säikeet, RAM:n tyypin (DDR5/DDR4) ja nopeuden, GPU:n ja VRAM:n käytön — kaikki yhdellä silmäyksellä. Tukee useita GPUita.

🎓

Autonominen opetustila

AI hakee itsenäisesti koodausdataa Stack Overflowsta, GitHubista, Real Pythonista ja HA Communitysta — salasanasuojattu, ajastettava, live-logi asetuksissa. Korjaukset muuttuvat automaattisesti uusiksi hakuaiheiksi.

🔍

Parannettu web-haku

AI hakee ensin, vastaa sitten — ei arvata faktoja. DuckDuckGo Search API korvaa epäluotettavan HTML-scrapingin. Tukee myös oman SearXNG-instanssin konfigurointia täydelliseen yksityisyyteen.

Toimintaperiaate

Central Core pakkaa parhaat ratkaisut Knowledge Packeiksi. Kaikki LOCAL NODEt päivittyvät — sinun AI:si kasvaa koko yhteisön onnistumisilla.

Yksityisyys

Sinun datasi ei koskaan poistu koneeltasi

Yhteisöoppiminen tapahtuu ilman yksityisyyskompromisseja. Koodisi anonymisoidaan AST-tasolla ennen lähetystä — rakenne säilyy, mutta identiteetti ei.

  • Raakadata ei koskaan poistu koneeltasi
  • AST-anonymisointi poistaa kaikki tunnisteet
  • Sinä valitset mitä synkronoidaan
  • Täysin offline-käyttö mahdollinen
  • Jokainen agenttitoiminto lokitettu ja toistettavissa

Roadmap

Rakennuspolku

Viidessä vaiheessa kohti omaa foundation modelia.

Phase 1 · Q1 2026 ✓

LOCAL AI — Yksityinen AI-assistentti

Chat, pysyvä muisti (SQLite + Qdrant), tiedostotyökalut, Coder- ja Debug-agentit, sessiohistoria, Settings-paneeli, reaaliaikainen laitemonitorointi (CPU/RAM/DDR5/GPU/VRAM), Tailscale-etäkäyttö. Täysin offline. v0.1.0–v0.1.50

Phase 2 · Q2 2026 ✓

TOOL & MULTI-AGENT — AI tekee oikeita asioita

Shell sandbox, Git, SSH/SFTP, policy engine, 10+ erikoisagenttia (Planner/Coder/Debug/Critic/Security/Swarm), Tauri .exe desktop app, autonominen opetustila, parannettu web-haku, GitHub CI/CD 4 alustalle. v0.1.51–v0.1.84

Phase 3 · Q3 2026 ⬤

POLISH & CENTRAL CORE — Laadun viimeistely + yhteisöoppiminen

7 kielen i18n ✓, kielikorjaus system promptissa ✓, chat-historian meta-tag-siivous ✓, AST-anonymisointi ✓, Knowledge Pack -jakelu ✓, 285 taitorekisteriä ✓ — Central Core odottaa Hetzner-palvelinta. v0.1.85–v0.1.97

Phase 4 · Q4 2026

CENTRAL CORE LIVE — Yhteisö oppii yhdessä

Hetzner CX22 -palvelimen käynnistys, Central Core tuotantoon (Docker + Nginx + SSL), federatiivinen knowledge sharing käyttäjien välillä, plugin marketplace, VSCode-laajennus julkaistu.

Phase 5 · 2027

SELF-IMPROVEMENT — Järjestelmä kehittää itseään

Federated learning, meta-oppiminen agenttikomboille, synthetic benchmark generator, Private Team Cloud, macOS + Linux natiivibuildit.

Phase 6 · 2028+

OWN MODEL — C0D3A1 foundation model

LoRA fine-tuning anonymisoidusta onnistumisdatasta. 7B → 13B → 32B. Oma malli, oma data, oma tulevaisuus. C0D3A1 ei enää tarvitse Ollamaa.
