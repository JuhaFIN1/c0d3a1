Tätä ei voi kytkeä pois Asetuksista — se on tietoinen tuotepäätös, ei tekninen rajoitus (ks. kohta "Miksi pakollinen?" alla).
- **Central Core -palvelin ei ole vielä käynnissä.** Anonymisoitu oppimisdata kertyy toistaiseksi vain paikalliseen jonoon (`~/.c0d3a1/sync_queue/`) ja lähetetään palvelimelle heti kun se avataan — sitä ennen mitään ei lähetetä verkon yli.
- Kerättävä data anonymisoidaan AST-tasolla: muuttuja-, funktio- ja luokkanimet korvataan yleisnimiillä. Henkilötietoja, salasanoja, tiedostopolkuja tai IP-osoitteita ei koskaan lähetetä.
- **Chattiin liitetyt tiedostot** (kuvat, PDF:t, koodi): käsitellään muistissa, EI tallenneta pysyvästi levylle liitteinä.
- **AI:n luoma media** (kuvat, videot, musiikki): tallennetaan paikallisesti (`~/.c0d3a1/generated/`), ei lähetetä minnekään.
- Katso tarkemmat tietosuojakäytännöt: [Privacy Policy](PRIVACY.md)

### 4. Privacy and Data

- All local data remains on your own device (`~/.c0d3a1/`)
- **User profile** (first name, city): stored locally, used by AI to personalize responses. Never transmitted.
- **Community learning is a mandatory core feature of the service, not opt-in.** Verified corrections, successful conversations, and learned skills are anonymized (AST-level) and automatically transmitted from every user to improve the shared Central Core. This cannot be disabled in Settings — it is a deliberate product decision, not a technical limitation (see "Why mandatory?" below).
- **The Central Core server is not live yet.** Anonymized learning data currently accumulates in a local queue only (`~/.c0d3a1/sync_queue/`) and will be transmitted the moment the server is deployed — nothing is sent over the network before then.
- Collected data is anonymized at the AST level: variable, function, and class names are replaced with generic identifiers. Personal information, passwords, file paths, and IP addresses are never transmitted.
- **Files attached to chat** (images, PDFs, code): processed in memory only, NOT permanently saved as attachments.
- **AI-generated media** (images, video, music): saved locally (`~/.c0d3a1/generated/`), never transmitted anywhere.
- See full details: [Privacy Policy](PRIVACY.md)

### 5. Why is community learning mandatory?

C0D3A1's long-term goal is a community-trained AI model of its own. This only works if every user participates — sharing is therefore an automatic default for everyone, not an opt-in feature. In return, all users benefit from the collectively learned knowledge (Knowledge Packs). Anonymization (AST-level, no personal data/paths/identifiers) is always on and cannot be bypassed.
